#### 導言 - Lead paragraph
在 React 應用中，表單驗證是一個常見的需求。

使用正則式可以方便地對輸入的資料進行格式驗證，以確保用戶輸入的資料是符合特定要求的。

函式組件中可以使用 React Hook From 搭配 Yup 達到快速驗證與好設置的目的，但它們並不支援 Class 組件使用，所以只能自行撰寫或找套件。

下面將透過實際範例演示如何在 Class 組件表單中運用正則式進行驗證，並於驗證不通過時提供相對應的錯誤提示。
